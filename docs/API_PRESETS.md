# Android API presets

The web UI includes built-in presets:

| Android | API | Codename | Targets |
|---|---:|---|---|
| Android 13 | 33 | Tiramisu | `google_apis`, `google_apis_playstore` |
| Android 14 | 34 | Upside Down Cake | `google_apis`, `google_apis_playstore` |
| Android 15 | 35 | Vanilla Ice Cream | `google_apis`, `google_apis_playstore` |
| Android 16 | 36 | Baklava | `google_apis`, `google_apis_playstore` |

The built-in list is for convenience. The real source of truth is still `sdkmanager`.
Use **Refresh SDK API/system-image list** in the web UI to cache the system images that are actually available for the installed command-line tools.
